サービス概要

指定された URL のページを印刷用 CSS を適用して PDF 化し、Xteink X3 向けの XTC ファイルに変換する、個人運営のサービスです。

PDFファイルのアップロードについて

お手元の PDF ファイルをアップロードし、ブラウザ内であらかじめ設定を確認したうえで XTC に変換できます。アップロードされた PDF は変換処理の完了後に速やかに削除され、R2 のライフサイクル設定により最長でも 1 日以内には自動的に削除されます（利用者には提供されません）。

次のものには対応していません。

  • パスワードで保護された PDF、暗号化された PDF
  • PDF 内テキストの再構成、フォント・文字サイズ・行間の変更、横書き・縦書きの再組版、OCR
  • 見開き分割、自動ノド除去、自動傾き補正、自動余白検出
  • 注釈・フォーム入力内容の編集、ページごとに異なる個別設定
  • サイズ上限を超えるファイルのアップロード、変換開始後のジョブキャンセル

TXTファイルのアップロードについて

お手元のテキストファイル（.txt）をアップロードし、ブラウザ内で文字コードの判定・本文プレビュー・組版設定（書字方向、フォント、文字サイズ、行間、余白など）を確認したうえで XTC に変換できます。アップロードされたテキストファイルおよび変換設定、変換途中に生成される中間ファイル（組版済みHTML・中間PDF）は、変換処理の完了後に速やかに削除され、R2 のライフサイクル設定により最長でも 1 日以内には自動的に削除されます（利用者には提供されません）。生成された XTC ファイルの保存期間は他の変換と同じく約 24 時間です。

「X3実機プレビューを生成」機能は、本文の先頭部分のみをサーバーへ送信してその場で XTC に変換し、変換結果をブラウザへ返す機能です。送信された本文・生成された HTML やフォントデータ・変換結果の XTC ファイルは、いずれもサーバー（R2）に一切保存されず、リクエストの処理が終わり次第破棄されます。

次のものには対応していません。

  • Markdown・HTMLタグの解釈（本文は常にプレーンテキストとして扱われます）
  • RTF、DOCX、EPUB などの他形式、OCR、画像埋め込み、外部画像・外部CSS・外部JavaScriptの読み込み
  • 青空文庫注記の解釈、ルビ・傍点・脚注・目次
  • UTF-16、EUC-JP、ISO-2022-JP などの文字コード
  • 複数テキストファイルの結合、ページ単位の個別書式、変換開始後のジョブキャンセル

プライバシー

制定日: 2026年7月18日／最終改定日・適用開始日: 2026年7月23日

本サービスが取得または保存する情報は、本人認証、アカウント・ライブラリ・端末配信機能の提供、利用上限の管理、不正利用の防止、セキュリティの確保、障害調査、問い合わせ対応および法令上必要な対応のために利用します。

アカウント登録は任意です。登録しなくても URL や PDF・TXT ファイルの変換は利用できます。本サービスは広告・マーケティング目的のアクセス解析を行っていません。アカウントを作成していない場合、変換履歴は利用者のブラウザ内にのみ保存されます。

パスキー（WebAuthn）でアカウント登録をすると、変換したページを『ライブラリ』に保存し、複数の Xteink X3 端末で共有できます。アカウント登録にメールアドレスやパスワードは必要とせず、保存する情報は次のとおりです。

  • 表示名（アカウント作成時に入力した名前。メールアドレスやパスワードは保存しません）
  • パスキーの認証情報（署名検証に使用する公開鍵等。秘密鍵は本サービスには送信・保存されず、利用者の端末または利用するパスキープロバイダーによって管理されます）
  • セッション情報（ログイン状態を保持する Cookie。有効期限は既定で30日、同時に有効なセッション数は1アカウントあたり最大10件です）
  • ペアリングした Xteink 端末の情報（端末名、ペアリング状態、その端末に配信するライブラリ項目の一覧）
  • ライブラリのメタデータ（タイトル・著者名などの書誌情報）
  • 変換済みの XTC ファイル本体

セッションは有効期限が切れると自動的に削除され、ログアウトなどで無効化されたセッションも一定期間後に削除されます。ライブラリのメタデータおよび XTC ファイル本体は、原則として、利用者が個別に削除するか、アカウントを削除するまで保持されます。ただし、利用規約違反への対応、サービスの変更・終了、セキュリティ上の対応その他利用規約に定める場合には、それ以前に削除されることがあります。新規登録時の不正防止のために一時的に記録する送信元 IP アドレスは、平文では保存されず復元できない形にハッシュ化したうえで保存され、7日以内に自動的に削除されます。

アカウントは、アカウント画面から確認のため「DELETE」と入力することでいつでも削除できます。削除すると、保存されていたライブラリのメタデータ（データベース）と XTC ファイル本体（ストレージ）を含め、アカウントに関する情報が削除されます。この操作は取り消せません。

また、セキュリティ、障害調査および変換処理のため、Cloudflare Workers、Workflows、R2、D1 等にアクセス情報や処理情報が一時的に記録される場合があります。

運営者は、法令に基づく場合、人の生命・身体・財産の保護に必要な場合、その他法令上認められる場合に、保存された情報を関係機関その他の第三者へ提供することがあります。

新規アカウント登録には Cloudflare Turnstile を利用しています。登録時には、ボット判定のため、IPアドレス、ブラウザおよび端末環境に関する情報その他の通信情報が Cloudflare に送信され、Cloudflare のプライバシーポリシー等に基づいて処理されます。

フォントの配信に Google Fonts を利用しているため、ページ表示時にブラウザから Google のサーバーへフォント取得のリクエストが送信されます。その際、IPアドレス、要求URL、ユーザーエージェント、参照元ページ等の情報が Google に送信されます。
